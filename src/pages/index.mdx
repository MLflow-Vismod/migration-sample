# <Delta Lake /> guide

[Delta Lake](https://delta.io) is an [open source storage
layer](https://github.com/delta-io/delta) that brings reliability to data lakes.
<Delta Lake /> provides ACID transactions, scalable metadata handling, and
unifies streaming and batch data processing. <Delta Lake /> runs on top of your
existing data lake and is fully compatible with <Apache Spark  /> APIs. <Delta Lake /> on
<Databricks> allows you to configure <Delta Lake /> based on your workload
patterns.
